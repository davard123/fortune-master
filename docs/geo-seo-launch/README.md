## ❓ FAQ（给团队的问题）

**Q：为什么 3 个站用同一个名字（David Liu）？**
A：AI 引用会优先抓真人专家。"David Liu 在 3 个不同的权威领域"远比"3 个假身份在 3 个主题"信号强。LinkedIn 的 E-E-A-T 也偏向真人一致身份。

**Q：为什么不一次性把所有内容都发？**
A：平台有"24h 第 3 篇"限流规则。批量发会被限 = 浪费内容。持续稳定是平台算法的最爱。

**Q：如果某平台的帖子被 AI 识别了怎么办？**
A：立刻删除 + 冷却 7-10 天。同时用更口语化版本重发。这是正常的部分——10-20% 的内容会被识别。

**Q：要外包给别人写吗？**
A：可以，但必须经"AI 写 → 真人改 → 内部交叉审"三步。Renting/loan 这种"专业建议"内容必须由有经验的人审过——平台 mods 和 AI 都会识破纯 AI 内容。
