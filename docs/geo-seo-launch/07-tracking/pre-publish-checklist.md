# 发布前最后检查清单 + 防删帖规则

> 每篇内容在发布前 5 分钟，过一遍这个清单。这是保住账号、留住 SEO 价值的最后一道防线。

---

## A. Medium 文章发布前清单

### 账号/Publication 层
- [ ] 用的是对应 Publication，不是个人 profile（除非只是第 1 帖试水）
- [ ] 作者署名"David Liu" + 主站 URL 在 bio 里
- [ ] Publication 描述已优化

### 文章质量层
- [ ] 标题含主关键词 + 具体数字/年份
- [ ] 字数 ≥ 1500（短于此 Medium 不推流）
- [ ] 至少 3 个 H2 子标题
- [ ] 至少 1 段 List / Numbered List
- [ ] 至少 3 个具体事实 / 数据 / 法规引用
- [ ] 至少 1 段 FAQ（3+ Q&A）
- [ ] Author bio + "Originally published on [URL]" 在文末
- [ ] 1-2 张原创图（不是 stockphoto）

### 频率 / 风险层
- [ ] 今天还没在该 Publication 第 3 篇（24h 限）
- [ ] 上一篇文章 ≥ 24 小时前发

---

## B. LinkedIn 长帖发布前清单

### 内容层
- [ ] 第 1 句是 hook（不是"我今天想分享...")
- [ ] 字数在 800-1500 字
- [ ] 段落短（1-2 行，移动端阅读）
- [ ] Hashtag 3-5 个（不能更多）
- [ ] 没有 emoji 堆叠（≤ 3-5 个合适）
- [ ] 至少 2 个具体数据 / 案例 / 数字
- [ ] 没有"as an AI"、"in today's fast-paced world"这类 AI 痕迹
- [ ] 没有外链在正文

### 发布层
- [ ] Profile 显示 100% 完成（头像、bio、Experience 完整）
- [ ] 发布时间：本地时间早 8-10 AM
- [ ] 帖子发布后 5 分钟内在评论区贴主站 URL 作为置顶评论

---

## C. Reddit 帖发布前清单（最重要！）

### 账号层（必先过）
- [ ] 账号 ≥ 14 天年龄（在目标 sub 内互动历史≥ 7-10 天）
- [ ] 在该 sub 至少有 3-5 个点赞 + 1-2 个有用评论的历史
- [ ] User Flair 已设置（[CA]、[Lender]、[Self]等）
- [ ] Karma ≥ 50（部分 sub 要求）

### 标题层
- [ ] 严格遵守 sub 标题规则（如 `[Landlord-US][CA]` 前缀必加）
- [ ] 不能纯营销腔（"Check out my article..." 不行）
- [ ] 不能问"why doesn't anyone talk about..."这种 AI 极常见句式
- [ ] 提问或经验分享切入
- [ ] 标题 ≤ 80 字

### 正文层
- [ ] 字数 200-800 字（不要短到一眼像丢进去）
- [ ] 第一人称 + 具体个人经历（如有）
- [ ] 至少 1 个具体数字 / 时间 / 金额
- [ ] 没有 "as an AI" / "in conclusion" / "Hope this helps" 这类 AI 印记
- [ ] 没有外链（外链放评论区第一条）
- [ ] 不堆 emoji（≤ 2-3 个合适）
- [ ] 文末加 1 个开放式 ask（"anyone else..."）

### 评论第一条模板
- [ ] 不直接贴 URL（"For reference, link: [URL]）。先说"let me know if not appropriate"
- [ ] 回复评论时，每个≥ 30 字
- [ ] 1 小时内不离开 sub

### 评论互动层
- [ ] 发帖后 30 分钟内自查 hot 帖是否有新评论
- [ ] 每个评论回复≥ 30 字
- [ ] **不在评论里推销任何服务**——这是 Reddit 的死罪

---

## D. Quora 回答发布前清单

### 内容层
- [ ] 第 1 段独立可抓（80-120 字）
- [ ] 中段有 list / numbered
- [ ] 有 FAQ 段
- [ ] 字数≥ 350
- [ ] 主关键词出现 2-3 次（自然而不是堆叠）
- [ ] 没有外链在正文里
- [ ] 没有过度营销腔

### 平台层
- [ ] 答的是 high-view + low-answers 问题，不是新发问
- [ ] 账号 ≥ 7-10 天养号
- [ ] Bio 已贴主站 URL
- [ ] 没有同日答 ≥ 3 条同一个主题

---

## E. 主站 SEO 内容同步检查（每个 Medium 文章发布后 30 分钟）

- [ ] 主站同步发布（同一篇）— 必要，但**内容必须做 30-40% 的本地化改写**——不能 100% 复制
- [ ] 主站用 WordPress / Webflow 加 Article Schema
- [ ] GSC → URL Inspection → Request Indexing
- [ ] GSC → Inspect the URL → confirm indexing started
- [ ] 主站内链：文末跳到主站其他相关文章
- [ ] 主站 URL 简短（slug 用英文短描述）

---

## F. 跨平台内容复用规则

### 可以复用（但必须本地化）
- 同一篇文章 → Medium（结构化版）→ LinkedIn（简化版）→ Quora（短答版）
- 不要 100% 复制粘贴
- 每个平台的语法习惯不同

### 不能复用
- Reddit 帖 vs LinkedIn 帖 → **不同**！Reddit 不可营销腔、Longform 太空
- Reddit 帖 vs Quora 回答 → Quora 可以更正式，Reddit 必须"我经历过"

### 跨平台自查表
| 平台 | 字数 | 语气 | 外链位置 |
|------|------|------|----------|
| Medium | 1500+ | 专业深度 | 文末 "Originally published on..." |
| LinkedIn | 800-1500 | 实战分享 | 评论区第一条 |
| Reddit | 200-800 | 真人第一人称 | 评论区第一条 |
| Quora | 350-800 | 行业专家 | bio |
| 主站 | 1500+ | SEO 优化版 | 内链 + 站内跳转 |

## G. 防删帖红线（全平台通用）

### ❌ 必删红线
- 标题含 URL（如 "Check out loaninca.com"）
- 第 1 段开头放联系方式或促销
- 同文复制 + 微调词序复制到 5 个平台
- 段落里有 "click here" / "sign up now" / "learn more"
- 账号 0 历史首日发
- 标题/正文用 emoji 占多数
- 配图是从 stockphoto 抠的、有版权痕迹
- 含 "I'm an AI" / "language model" / "as an AI assistant"

### ⚠️ 高风险红线（不同平台不同容忍度）

| 平台 | 高风险行为 | 容忍度 |
|------|----------|--------|
| Reddit | 自介里含多个站点链接 | 低 |
| Reddit | 1 周内 ≥ 2 帖同 sub | 中 |
| Reddit | 评论全是"great point"短句 | 中 |
| LinkedIn | 引用某个 site 的 mention 超过 3 次 | 中 |
| LinkedIn | 纯转载别人内容带自己 CTA | 低 |
| Quora | 完全相同答案回不同问题 | 低 |
| Medium | 同一账号 24h 第 3 篇 | 即时限流 |
| Medium | 短文（< 500 字）大量发 | 长期限流 |

## H. 发布后的 48 小时管理

### 第一个 30 分钟
- 立即回复前 3-5 条评论（评论区互动越快，平台越推流）
- 至少 1 个评论是 ≥ 50 字的深度回复
- 不要离开这个平台的核心 sub

### 第 2-24 小时
- 监控是否有 downvote / 删帖
- 至少 2 次回到帖下互动
- 1 个"自我顶贴"——发新评论说"想跟进哪些朋友有具体问题？"

### 第 24-48 小时
- 看平台数据（impressions / clicks / view count）
- 如有 downvote 持续增加 → 移除 + 反思（48h 后再发新）
- 如有 30+ upvote + 真评论 → 在评论里继续问与帖子主题相关的深入问题，引导讨论

### 第 7 天
- 看长期数据
- 写 KPI 表
- 决定下一篇主题（吸收反馈）

## I. 紧急处置预案

### 账号被封
| 平台 | 申诉方式 | 备选方案 |
|------|---------|---------|
| Medium | email → support@medium.com | 用新邮箱开新账号（每周 5 篇限制，但**别同时开多个**） |
| LinkedIn | 账户设置 → Help → Restrictions | 用备份 profile（推荐 W1 就建 1 个） |
| Reddit | /r/redditappeal 或 mods DM | 在其他 sub 重新培养新账号 |
| Quora | appeal from account email | 同上 |

### 内容被删
- 看是否有 mod 留言？→ 修改 → 重发
- 没有 mod 留言？→ 30 天后改 1 字重发
- 2 次被删同 1 篇 → 换 sub 或换表达角度

### 内容被 downvote 围攻
- 不回应 → 等 24h 看是否平息
- 没有恶化 → 不要补评论
- 仍恶化 → 删帖（内容仍存活在你自己草稿）

### GSC 发现 negative SEO 攻击
- 立即查 Ahrefs Webmaster → "Disavow" → 提交 disavow 文件
- 检查是否有 spam 外链（看域名后缀是否有 .ru / .cn 或不相关）
- 提交"Reconsideration Request" 到 Google

## J. 最后给你一个"心理预期"

> **做这套 SOP 之前先接受这 3 件事**：

1. **前 2 个月不会看到稳定排名**——这是行业的客观规律，不是不努力
2. **Reddit 会有删帖**——会发生，不要怀疑努力，可能就是账号还太新
3. **AI 引用是慢热指标**——需要 6-12 周才有稳定结果
4. **KPI 在 W12 前不会"漂亮"**——这正常，看到趋势对就对了

W12 之后你们应该已经在：
- 主站有 20+ 条高权重外链
- 几个核心关键词进 Google page 2
- AI 引用触发在 5+ 个 prompt
- 几个月内累计到几个稳定的"被 AI 引用"的内容

W24 之后你才会看到：
- 主关键词进 page 1 排名
- 有机流量稳定增长（≥ 1K/月）
- AI 引用触发率 ≥ 15%

如果不接受这个预期，请考虑付费 SEO 服务或考虑 6 个月以上的活动。
