# 今日发布 / 互动检查表（一页纸模板）

> 这是给 W2 开始后,每个发布日使用的速查表打印版。

---

## 周一(发布日)

### ☐ 08:30 发布第 1 篇 Medium 文章
- [ ] 在 Medium Publication(loaninca / rentalinca / fopusha 对应版)粘贴内容
- [ ] 选原文 publish,不允许 rewrites
- [ ] 配图:1 张原创(custom chart 或 screenshot)
- [ ] 文末加 "Originally published on [URL]"
- [ ] 作者 Bio: David Liu + 主站 URL
- [ ] 发布后等 5 分钟,文章在 Medium 主页可见
- [ ] 把 Medium 文章 URL 写到本表

```
Medium URL 1: https://medium.com/[pub-slug]/[article-slug]_______________
```

### ☐ 09:00 同步到主站
- [ ] 把文章 30-40% 改写后粘贴到主站
- [ ] 嵌 Article Schema(参考 08-site-assets/schemas.md)
- [ ] 嵌 FAQ Schema(每篇文章末尾都有 FAQ,提取生成 schema)
- [ ] Google Search Console → URL Inspection → Request Indexing
- [ ] 主站 URL 写入本表

```
主站 URL 1: https://[site].com/[slug]_______________
```

### ☐ 09:30 LinkedIn 发长帖
- [ ] 复制 linkedin-posts 对应版本
- [ ] 检查 hashtag 3-5 个
- [ ] 检查 hook(第 1 句)
- [ ] 发布
- [ ] 立刻评论区贴 Medium 文章 URL 作为置顶

```
LinkedIn URL 1: https://linkedin.com/posts/david-liu-____-_____/_______________
```

### ☐ 10:30 LinkedIn / Medium 互动 30 分钟
- [ ] LinkedIn 自评论回 5 个真评论
- [ ] Medium 看是否有新 comments
- [ ] LinkedIn 给 2-3 个相关行业 KOL 评论 100 字

---

## 周二(Reddit 日)

### ☐ 08:30 检查 Reddit 养号状态(只在新发布日检查)
- [ ] 账号龄 ≥ 14 天
- [ ] 累计 ≥ 20 个有效评论
- [ ] User flair 已设置

### ☐ 09:00 发 Reddit 主帖
- [ ] 复制 reddit-posts 中"真人口语版"(不是结构化版)
- [ ] 检查 sub 标题前缀(`[Landlord-US][CA]` 这类)
- [ ] 没有外链在正文
- [ ] 没有"as an AI"痕迹
- [ ] 发布后 5 分钟内评论第一条

```
Reddit URL 1: https://reddit.com/r/[sub]/comments/[id]/_______________
```

### ☐ 09:30-11:30 Reddit 互动(2 小时)
- [ ] 不能离开 sub(会 lost engagement)
- [ ] 每条评论 50 字+回复
- [ ] 至少 5 个深度回复

---

## 周三(Quora 日)

### ☐ 08:00 找目标问题
- [ ] 在 Quora 搜索主关键词
- [ ] 按 Most Viewed 排序
- [ ] 找 "View 多 + Answer 少" 的
- [ ] 复制已写 quora-answers.md 中匹配的回答

### ☐ 09:00 发布回答
- [ ] 进入目标问题
- [ ] 粘贴答案
- [ ] 第 1 段独立可抓
- [ ] 中段 list
- [ ] FAQ 段在末尾
- [ ] bio 不在答案内(已经在 profile bio 里)
- [ ] 发布

```
Quora Answer URL 1: https://quora.com/[question]/answer/_______________
```

### ☐ 09:30 Quora 同主题找机会
- [ ] 再答 1-2 个同主题相关老问题
- [ ] 必须差异化内容(不能直接复制自己之前的回答)

---

- [ ] 不要广告腔

### ☐ 09:00 Reddit 同 sub 互动(没有发新帖)
- [ ] 评论 +3-5 个帖子

- [ ] 每条 30 字+

### ☐ 10:00 监控各平台
- [ ] 查看昨日 LinkedIn 帖表现
- [ ] 检查 Medium/LinkedIn 互动数据
- [ ] Reddit 帖 vote 数(看是否触发 AutoMod 删除)

## 紧急处理

### 帖子被删
- [ ] 立即 stop 发
- [ ] 24h 冷却后再考虑重发
- [ ] W5+ 才重发相同 sub 同主题

### 账号被 ban
- [ ] 立即停止所有动作
- [ ] 申诉(email 平台)
- [ ] 7-14 天后再操作新账号

### 流量异常下降
- [ ] 不恐慌
- [ ] 查 GSC / Ahrefs data
- [ ] 查是否有违规邮件
