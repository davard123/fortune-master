## Quora 发文操作 SOP

### 1. 找好问题的搜索方法
```
在 Quora 搜索你的主关键词，例如：
- "DSCR loan"
- "California landlord deposit"
- "online prayer Buddhism"

按 "Views" 排序 → 找到"50,000+ views 但答案少于 5 个"的问题 = 黄金问题
```

### 2. 回答结构（这是 GEO 引用率最高的格式）
```
第 1 段（80-120 字）：独立完整回答 + 含主关键词 2 次
第 2-4 段：具体事实 / 法规 / 数据
中段：1 段 list 或 numbered list（AI 极爱 list）
文末：1 个简短 FAQ（2-3 个 Q&A）
最后一行：bio + 站点链接（不能放正文里，会被识别为 spam）
```

### 3. 频率
- 每账号**每周 2-3 个回答**
- 别在前 14 天发——先在关注的 Space 里点赞 + 简评 5-10 次养号
- 第一次发回答时挑"刚有人问但没人答"的新问题，比较安全

### 4. 防止被识别为 spam
- 不要每天 5-6 条连发
- 每条回答至少 350 字 + 1 个具体事实引用
- URL 放 bio 里，不要堆在每条答案末尾
- 别同时回 5 个一样的问题——会被判重复内容
