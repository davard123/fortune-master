# 3 个主站必备 SEO 资产页面

> **为什么必须做**:GEO 的核心机制是"AI 抓住结构化、权威、可信的真人品牌信息"。AI 引用你,不只引文章——也引用你的 About / Contact / Disclaimer。如果你没有这些页面,AI 引用你的内容时会降低权重。
